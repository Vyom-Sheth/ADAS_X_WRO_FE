This is a github repository for the World Robotics Olympiad Future Engineer's category.
The participants are Vyom Sheth and 
We have made a car which has 4WD with one shaft.
The motor is of 10000RPM
The main controller is Raspberry Pi 4 Model B
The subcontroller is Arduino Uno.
The motor driver is L298N.
The battery is of 7.4V.
Python is used as our main programming language and for visual processing as well.
Picamera V2 is used as the camera.
